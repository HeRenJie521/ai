# 智蚁 AI 平台 - 功能清单月报

**报告周期**：2026 年 4 月  
**项目名称**：智蚁 AI 对话平台  
**技术架构**：Vue 3 + Spring Boot

---

## 一、项目概述

智蚁 AI 是一个企业级智能对话平台，可以理解为"企业定制的 AI 客服系统"。它让企业能够快速搭建自己的 AI 助手，并嵌入到官网、APP 等各类系统中。

---

## 二、核心功能模块

### 📱 1. 用户端功能（普通用户可见）

| 功能 | 说明 | 完成度 |
|------|------|--------|
| **智能对话** | 用户可与 AI 进行多轮对话，支持文字 + 图片输入 | ✅ |
| **思考过程展示** | 能看到 AI 的"思考过程"（类似草稿纸），理解 AI 如何得出结论 | ✅ |
| **历史会话管理** | 自动保存聊天历史，可随时查看、删除、新建会话 | ✅ |
| **多模型切换** | 支持切换不同的大模型（如文本模型、深度思考模型、视觉模型） | ✅ |
| **图片上传** | 支持上传图片让 AI 识别（如截图、照片等，最大 5MB） | ✅ |
| **开场白与推荐问题** | 进入应用时显示欢迎语和常见问题，引导用户提问 | ✅ |
| **嵌入网站聊天** | 可嵌入企业官网，访客无需登录即可对话 | ✅ |

## 三、技术亮点

### 🔐 安全特性
- JWT Token 认证机制
- API Key 鉴权（支持请求头方式）
- HMAC 签名验证（嵌入网站 SSO）
- CORS 跨域安全配置
- 管理员权限隔离

### ⚡ 性能优化
- SSE 流式输出（打字机效果）
- 异步线程池处理聊天请求
- Redis 缓存登录会话
- 分页查询优化

### 🎨 用户体验
- Markdown 渲染（支持代码高亮）
- 流式打字机效果
- 图片上传预览
- 响应式布局（支持 PC/移动端）
- 思考过程折叠展示

## 五、前后端技术栈

### 后端（eaju-ai-api）
- **框架**：Spring Boot
- **数据库**：MySQL
- **缓存**：Redis
- **安全**：Spring Security + JWT
- **HTTP 客户端**：Java HttpClient

### 前端（eaju-ai-web）
- **框架**：Vue 3 + TypeScript
- **UI 库**：Naive UI
- **状态管理**：Pinia
- **路由**：Vue Router
- **HTTP 客户端**：Axios
- **Markdown 渲染**：markdown-it / highlight.js
