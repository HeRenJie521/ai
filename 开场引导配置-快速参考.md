## 📝 配置示例

### 管理端操作流程
1. 进入"集成管理"页面
2. 找到类型为"嵌入网站"的集成
3. 点击操作列的"开场白"按钮
4. 填写：
   - 开场白文本：`你好，我是 AI 助手，有什么可以帮助您的吗？`
   - 推荐问题：逐条输入，按 Enter 添加
     - `产品有哪些功能？`
     - `如何收费？`
     - `技术支持联系方式`
5. 点击"保存"

### API 响应示例
```json
{
  "welcomeText": "你好，我是 AI 助手，有什么可以帮助您的吗？",
  "suggestions": ["产品有哪些功能？", "如何收费？", "技术支持联系方式"]
}
```
