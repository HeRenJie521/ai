## 🚀 部署步骤

### 1. 执行数据库迁移
```sql
-- 连接数据库后执行
\i db/08-api-key-welcome-config.sql
```

### 2. 部署后端
```bash
cd eaju-ai-api
mvn clean package
```

### 3. 部署前端
```bash
cd eaju-ai-web
npm install
npm run build
```

## 📝 配置示例

### 开场白配置
```
开场白文本：你好，我是 AI 助手，有什么可以帮助您的吗？
推荐问题：["产品有哪些功能？", "如何收费？", "技术支持联系方式"]
```

### API 响应示例
```json
{
  "welcomeText": "你好，我是 AI 助手，有什么可以帮助您的吗？",
  "suggestions": ["产品有哪些功能？", "如何收费？", "技术支持联系方式"]
}
```

## ✅ 功能特性

1. **显示条件**
   - ✅ 无对话记录时显示
   - ✅ 已配置开场白文本
   - ✅ 仅对嵌入网站类型生效

2. **交互体验**
   - ✅ 点击推荐问题自动发送
   - ✅ 防重复点击
   - ✅ 加载动画反馈
   - ✅ 点击后隐藏开场引导

3. **配置灵活**
   - ✅ 可仅配置开场白
   - ✅ 可仅配置推荐问题
   - ✅ 两者都可有可无
