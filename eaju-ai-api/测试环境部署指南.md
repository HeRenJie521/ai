# 测试环境部署指南（简化版）

## 📋 配置说明

测试环境配置已写死在 `application-test.yml` 中，无需额外配置环境变量。

---

## 🚀 部署步骤

### 1. 编译项目

```bash
cd eaju-ai-api
mvn clean package -DskipTests
```

### 2. 启动应用（使用 test profile）

```bash
java -jar target/eaju-ai-api-1.0.0-SNAPSHOT.jar --spring.profiles.active=test
```

### 3. 后台运行（推荐）

```bash
nohup java -jar target/eaju-ai-api-1.0.0-SNAPSHOT.jar --spring.profiles.active=test > app.log 2>&1 &
```
